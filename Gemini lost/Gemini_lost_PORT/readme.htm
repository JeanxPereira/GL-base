Installation Instructions

PC

1.      Double-click on the downloaded Gemini Lost installation executable file to begin the installation program.

2.      Follow the instructions in the installation program to install Gemini Lost.

3.      After installation, double-click on the Gemini Lost desktop shortcut, or by using the appropriate shortcut in the Start Menu.

4.      To uninstall Gemini Lost, use the appropriate shortcut in the Windows program menu or by accessing Windows’ Add/Remove Programs control panel.

MAC

1.      Double-click on the downloaded Gemini Lost installation executable file to begin the installation program.

2.      Follow the instructions in the installation program to install Gemini Lost.

3.      After installation, double-click on the Gemini Lost icon in your Applications folder.

4.      To uninstall drag the Gemini Lost icon from the Applications folder into your trash.

How To Play

Characters:

·         Drag ‘n drop characters to assign to tasks

·         Hold a character over various locations in the map to see what work they can do there

 

Marriage:

·         Drag any unmarried castaway to the Chapel and ring the bell to bring all villagers to the Chapel

·         Find another castaway with a compatible zodiac sign and drag them to the Chapel to have them marry and reproduce

 

Map/Mini-Map:

·         Click anywhere on the Map/Mini-Map to instantly go to that location

·         On the Mini-Map yellow dots represent idle people, blue dots represent working people, teal dots represent children, and red dots represent sick people.
